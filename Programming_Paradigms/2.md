### type system && represent

#### tips 
   - 1 byte = 8 bit
   - int,float 4 byte
   - short 2 byte
   - char 1 byte
   - 负数用反码(取反末位加1)，便于相加为0，便于减法

#### type conversion
   - int to short， 取后16位
   - char to int 转出ascii码的位数， eg: A to 65, int to char 类似 
   - float to int 只能保留小数点前的7位

#### float
- 后面23 bit represent 小数点的后的数字
